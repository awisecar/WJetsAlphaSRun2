
*** Analyzing the FastNLO tables on the NEU Tier3:
1.  rename your zeroth table to this format: d**-x01-y01-0000.tab
2.  gzip all of your tables (“gzip *.tab”)
3.  source ~/MakeTablesv3/sourceEnvV3.sh ("sourceEnvV3.sh" is a copy of the shell script generated by the installation that sets up the environment needed for Sherpa, Rivet, etc.)
4.  run doMergeTables.py (have to move the python file to the same directory as the tables)
5.  export LHAPDF_DATA_PATH=<home directory>/localv3/share/LHAPDF:$LHAPDF_DATA_PATH
6.  run getStatUncert.py
7.  cmsenv within ~/CMSSW_7_1_30/src
8.  export LHAPDF_DATA_PATH=<home directory>/localv3/share/LHAPDF:$LHAPDF_DATA_PATH
9.  run getASVarFNLO_ME.py
10. run getASVarFNLO_MEPDF.py
A.  can also run getPDFUncert.py, getScaleUncert.py (needs newer version of FastNLO, use installation v2; source the environment inside the MakeTablesv2 folder)

*** Miscellaneous:
If you want to do NP corrections (calculated using Sherpa), need to use the yodamerge script:
1. source the local FNLO environment
2. cmsenv in CMSSW_10_0_0/src

*** Fastnlo-Rivet Closure Test:
set environment:
	source ~/MakeTablesv3/sourceEnvV3.sh 
	cmsenv in ~/CMSSW_10_0_0/src
then run command, e.g.:
	./getClosureRivetFNLO.py d56-x01-y01 --yoda=w1jetsFO_52M_9.yoda --pdf=CT10nlo --out=closureRivetFNLO_w1j_d56.png
